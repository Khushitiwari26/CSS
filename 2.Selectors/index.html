Selectors in HTML

Element Selectors

The element selectors selects HTML elements to select a specific element.

Id Selector

The id selectors uses the id attribute of an HTML element to select a specific element.The id of an element is unique within a page,so the id used to select one unique element!

Class Selector

The class selector selects HTML elements with a specific class attribute.To select elements with a specific class,write a period(.) character,followed by the class-name.
